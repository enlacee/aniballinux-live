# 🖱️ Mouse Overlay Linux

Círculo visual para el mouse en Linux (X11) ideal para grabar tutoriales.

## ✨ Features
- Sigue el cursor en tiempo real
- Cambia de color al hacer click
- Overlay transparente
- Ligero y rápido

## ⚙️ Instalación

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python mouse_overlay.py
```
## Requisitos

- Linux (X11)
